# Text_to_Speech using python gTTs
A way to turn your text into speech using gTTs

# The requirements
python 3.6.9

gTTS 2.1.1

pip 20.0.2
#
Installing python

$ sudo apt-get update

$ sudo apt-get install python3.6.9
#
Installing pip3 

if python-v<3, then use "python" instead of "python3"

$ python3 get-pip.py
#
Installing gTTs

$ pip3 install gTTS
